Protocol Buffer compatibility.

Protocol Buffer primitive types are mapped to AIngle types as follows:
protobuf typeAIngle type
int32, uint32, sint32, fixed32, sfixed32int
int64, uint64, sint64, fixed64, sfixed64long
floatfloat
doubledouble
boolboolean
stringstring
bytesbytes
enumenum
messagerecord

Notes:

  • protobuf repeated fields are represented as AIngle arrays
  • protobuf default values are translated to AIngle default values
