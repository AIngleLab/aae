Thrift compatibility.

Thrift primitive types are mapped to AIngle schemas as follows:

Thrift typeAIngle schema
bool"boolean"
byte{"type": "int", "thrift": "byte"}
i16{"type": "int", "thrift": "short"}
i32"int"
i64"long"
double"double"
string"string"
binary"bytes"

Thrift complex types are mapped to AIngle complex types as follows:

ThriftAIngle
structrecord
enumenum
listarray
setarray (Note: "thrift":"set" is added to schema.)
mapmap (Note: only string keys are permitted.)
