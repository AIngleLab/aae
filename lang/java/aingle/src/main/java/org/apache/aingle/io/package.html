Utilities for Encoding and Decoding AIngle data.

{@link org.apache.aingle.io.Encoder} is the abstract type encapsulating the encoding of AIngle data. Encoder instances are created and configured with {@link org.apache.aingle.io.EncoderFactory}.

{@link org.apache.aingle.io.Decoder} is the abstract type encapsulating the decoding of AIngle data. Decoder instances are created and configured with {@link org.apache.aingle.io.DecoderFactory}.
