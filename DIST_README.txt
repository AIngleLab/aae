Apache AIngle Distribution 

AIngle is a data serialization system.

This distribution contains the following files:

  - aingle-src-x.y.z.tar.gz contains the full source for AIngle, including
    all programming language implementations, documentation source, etc.

  - aingle-doc-x.y.z.tar.gz contains AIngle's pre-built documentation.

  - the c/, cpp/, csharp/, java/, js/, perl/, php/, py/, and ruby/
    subdirectories contain pre-built, language-specific binaries,
    bundles, etc. as conveniences.
